## Internship Domain – Data Analytics
**From Raw Data to Real Insights: Your Path to Data Analytics Mastery**

This repository contains my internship projects under the Data Analyst track, divided into three stages:

### Foundational Stage
1. **Sales Data Analysis**  
   - Tools: Python, Pandas, Matplotlib, Seaborn  
   - Insights: Category-wise sales, top products, regional performance  

2. **Student Performance Dataset**  
   - Tools: Python, Pandas  
   - Insights: Subject averages, top performers, validation checks  

3. **COVID-19 Data Report**  
   - Tools: Python, Pandas, Plotly/Power BI  
   - Insights: Daily cases, recoveries, deaths, trend visualization  

---

### Intermediate Stage
1. **E-Commerce Dataset Analysis**  
   - Tools: Python, Pandas, Seaborn  
   - Insights: Customer orders, revenue trends, best-selling products  

2. **Bank Customer Churn Analysis**  
   - Tools: Python, Pandas, Seaborn, Correlation Analysis  
   - Insights: Factors influencing churn, age/tenure/activity patterns  

3. **Interactive Sales Dashboard**  
   - Tools: Power BI / Tableau / Plotly  
   - Insights: KPIs (Revenue, Profit, Orders), filters by region/time/product  

---

### Advanced Stage
1. **Retail Sales Forecasting**  
   - Tools: Python, Pandas, ARIMA/Prophet  
   - Insights: Seasonal trends, demand forecasting, inventory recommendations  

2. **Customer Segmentation (Clustering)**  
   - Tools: Python, Pandas, Scikit-learn (K-Means)  
   - Insights: High Spenders, Loyal Customers, Discount Seekers, Low Engagement Users  

3. **Social Media Sentiment Analysis**  
   - Tools: Python, Pandas, NLP (TextBlob/VADER)  
   - Insights: Sentiment distribution, word clouds, trend analysis
